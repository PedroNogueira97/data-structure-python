## REGRAS DA ARVORE

1. Cada nó é vermelho ou preto
2. A raiz da árvore é preta
3. Todas as folhas são pretas (os nós representados com referência None)
4. Se um nó for vermelho, então seus dois filhos serão pretos
5. Não pode haver dois nós vermelhos adjacentes. Um nó vermelho não pode ter um pai ou filho vermelho
6. Todo caminho de um dado nó para qualquer um de seus descendentes (folhas None) contém o mesmo número de nós pretos
